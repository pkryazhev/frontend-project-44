<h1>Игры разума</h1>
<h2>Описание</h2>
<p>Данный проект содержит несколько консольных арифметических игр</p>
<h2>Минимальные требования</h2>
<ul>
    <li>Актуальная версия node.js</li>
</ul>
<h2>Установка и запуск</h2>
```bash
npm link
brain-even
```

<h2>Доступные игры</h2>
<h3>Brain-calc</h3>
<p>Задача игрока - правильно посчитать арифметическое выражение. Всего в игре 3 раунда, в случае ошибки игра заканчивается</p>
